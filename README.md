# Smart College Event Tracker (MongoDB)

A smart academic event tracking system for colleges with MongoDB storage. Key features:
- Shows only relevant academic events per student (department, semester, registered events).
- QR-based event attendance stored in DB for internal marks, NSS/club hours.
- Auto-generated participation certificates.
- Admin panel for faculty/HOD to manage seminars, workshops, circulars, deadlines.
- Student-specific notifications and exam-related reminders.
- Participation analytics and annual report export for NAAC.
- Event clash prevention and unified academic calendar view.
